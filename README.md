# 🍎 NYC Schools Data Analysis Portfolio

## 📌 Project Overview
This repository showcases a complete end-to-end data engineering and analysis workflow using NYC Open Data. 
The project covers everything from initial data exploration to SQL database integration and visualization.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, SQLAlchemy, Matplotlib, Seaborn
- **Database:** SQLite / PostgreSQL
- **Environment:** Jupyter Notebooks
